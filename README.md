## 프로젝트 소개

- 🦁 멋쟁이 사자처럼 프론트엔드 4기 바닐라 자바스크립트로 진행한 팀 프로젝트입니다.
- 🛒 제공된 시안을 토대로 요구사항을 정의하고 팀원들과 토의하여 백로그를 작성하여 작업하였습니다.
- 🛍️ 본 프로젝트는 자바스크립트만 사용하여 마켓컬리 디자인을 클론코딩한 프로젝트입니다.
